# SwiftTDF

SwiftTDF is a complete Swift implementation of TheDraw Font (.TDF) file format. Reads, writes, renders.

## Status

Under construction. The library will be fleshed out in subsequent commits (parser, writer, renderer, tests).

## Reference

The authoritative format reference ships with the repo: [Reference/TDF_SPECIFICATION.md](Reference/TDF_SPECIFICATION.md).

## Credits

The reverse-engineered .TDF specification is the work of Roy/SAC, originally published at https://www.roysac.com/blog/2014/04/thedraw-fonts-file-tdf-specifications/. SwiftTDF treats that document as the source of truth.
